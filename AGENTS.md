# TeamTigrr landing page Agent Rules

## Core Tech Stack
- **Frontend:** Single-page HTML5 with Tailwind CSS (CDN).
- **Styling:** Custom brand colors: `tigrr-orange` (#FF6B00), `tigrr-dark` (#111111), `tigrr-charcoal` (#1F1F1F).
- **Typography:** `Inter` (sans) and `Teko` (display).
- **Icons:** Font Awesome 6.4.0.

## Design Principles
- **Aesthetic:** Dark, high-contrast, modern "sports app" feel.
- **Components:** Use `clip-slant` and `clip-slant-reverse` for section transitions.
- **Responsiveness:** Mobile-first design; ensure all interactive elements work on touch.
- **Interactive:** Maintain hover states and smooth scrolling (`scroll-smooth`).

## Development Rules
- **Utility First:** Prefer Tailwind utility classes over custom CSS.
- **Vanilla JS:** Keep scripts minimal and contained within `index.html`.
- **Form Handling:** Lead capture submits to the Google Apps Script endpoint.
- **No Dependencies:** Avoid adding heavy libraries; keep the project lightweight and fast-loading.
- **Accessibility:** Use semantic HTML and ensure contrast ratios for the orange-on-dark theme.

## Content Voice
- Professional yet energetic and encouraging for junior sports clubs.
